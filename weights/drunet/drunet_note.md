In this folder, you are supposed to have the trained DRUNet weights registered as a file **drunet_3d_complex_denoise.pth**. It was too heavy to be uploaded here in github. But after having cloned this repository, you can download those weights through the link [huggingface.co/deepinv/drunet_3d_denoise_complex](https://huggingface.co/deepinv/drunet_3d_denoise_complex/tree/main), and put the file at this exact location. You are then all set for the reconstructions!
